Задание №3

Написать программу, экспортирующую заданный каталог в файл формата XML.

Например, на вход задается строка вида 
"C:\".
Процедура должна, начиная с этого каталога, вывести древовидную структуру 
всех подкаталогов в текстовый файл формата XML. Файлы можно не выводить, только каталоги.
Структура XML - на Ваше усмотрение.

Подсказки : 
1)Код будет работать только под WIN32, совместимость с другими не нужна
2)Можно использовать функции Shlwapi.dll и ShFolder.dll3)
3)XML можно создать не используя парсер, просто выводя строки в файл.
